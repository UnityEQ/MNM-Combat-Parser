How It Works

Three steps from download to damage numbers.

1

Download & Run

Grab ZekParser.exe from the download link. No install needed — it's a single portable file. Right-click and run as Administrator (required for network capture).

2

Start Playing

Launch the game as usual. ZekParser auto-detects the game process, finds the network connection, and starts reading combat data. Your player name is detected automatically.

3

Watch the Numbers

The combat feed scrolls in real time as you fight. The overview panel ranks everyone by total damage. Expand any player to see their ability breakdown. Copy the table to share with your group.
