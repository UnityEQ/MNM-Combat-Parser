SaltyVision

A crowdsourced item, experience, and combat database built from real play sessions. Every ZekParser user contributes data automatically — the more people parse, the more complete the database becomes.

What is SaltyVision?

The game doesn't publish loot tables or NPC stats. So we're building them ourselves.

SaltyVision is a living database powered entirely by the ZekParser community. As players run ZekParser during their sessions, combat events, item drops, NPC spawns, and experience gains are anonymously collected and aggregated into a searchable public database.

No manual data entry. No wiki editing. Just play the game with ZekParser running and your session data feeds into SaltyVision automatically. The more players contribute, the more accurate and complete the data becomes — drop rates get refined, XP tables fill in, and NPC combat profiles emerge from thousands of real encounters.

How It Works

Zero effort from you. Just play.

1

Play with ZekParser Running

ZekParser already captures every combat event, item drop, and XP gain during your session. With the API enabled, that data is sent to SaltyVision's servers in the background.

2

Data Gets Aggregated

Your session data is anonymized and merged with contributions from every other ZekParser user. Item stats are verified across multiple sightings. Drop rates converge toward true values as sample sizes grow. NPC profiles are built from hundreds of encounters.

3

Search the Database

Browse or search the full database on this site. Look up any item, NPC, or zone. See drop tables, XP yields, combat stats, and more — all backed by real data from real sessions.

Privacy & Data

SaltyVision collects game data only — item stats, NPC encounters, XP events, and loot drops. No personal information, no IP logging, no account credentials.
